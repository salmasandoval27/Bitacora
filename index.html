Lenguaje de etiquetas

Lenguaje que se utiliza para definir elementos en un documento o página web mediante la incorporación de códigos (etiquetas) rodeados de caracteres de inicio y fin comunes . Por ejemplo, las etiquetas en SGML, HTML, XML y WML utilizan el símbolo menor que para iniciar la etiqueta y un símbolo mayor que para finalizarla.

  • Hipertexto: Es texto que contiene enlaces a otros textos, tratándose así de una estructura no secuencial que permite crear, agregar, enlazar y compartir información de diversas fuentes por medio de enlaces asociativos y redes sociales.
  • DOM: Es una interfaz de programación para los documentos HTML y XML. Cuando nos referimos al DOM nos referimos a esta estructura de árbol, mediante la cuál podemos acceder a ella y modificar los elementos del HTML desde Javascript
  • Retroalimentación: Permite a las empresas crear sitios centrados en el usuario, que responden a sus expectativas y mejoran la experiencia general.
  • Interactividad: Se refiere a la capacidad de los usuarios para interactuar con un sitio web y sus elementos a través de diferentes acciones.
